# Required Resources

Use the listed sections needed for the weekly objective; do not complete every resource cover to cover.

## Phase 0 - Orientation

- [Google: Introduction to Machine Learning](https://developers.google.com/machine-learning/intro-to-ml)
- [Google: Problem Framing](https://developers.google.com/machine-learning/problem-framing)
- [PyTorch: Complete ML Workflow](https://docs.pytorch.org/tutorials/beginner/basics/intro.html)

## Phase 1 - Math and statistics

- [MIT 18.06SC Linear Algebra](https://ocw.mit.edu/courses/18-06sc-linear-algebra-fall-2011/) - vectors, matrices, projections, least squares, eigenvalues, and SVD.
- [MIT 6.041SC Applied Probability](https://ocw.mit.edu/courses/6-041sc-probabilistic-systems-analysis-and-applied-probability-fall-2013/) - probability, random variables, expectation, conditioning, and inference.
- [MIT Matrix Calculus for Machine Learning](https://ocw.mit.edu/courses/18-s096-matrix-calculus-for-machine-learning-and-beyond-january-iap-2023/) - selected lectures on gradients and matrix derivatives.
- [NumPy User Guide](https://numpy.org/doc/stable/user/) - arrays, broadcasting, linear algebra, random sampling.

## Phase 2 - Classical ML

- [Google Machine Learning Crash Course](https://developers.google.com/machine-learning/crash-course/) - complete modules through production ML and fairness.
- [scikit-learn User Guide](https://scikit-learn.org/stable/user_guide.html) - model selection, pipelines, preprocessing, metrics, calibration, and inspection.
- [scikit-learn Common Pitfalls](https://scikit-learn.org/stable/common_pitfalls.html) - leakage, inconsistent preprocessing, and randomness.

## Phase 3 - Deep learning

- [PyTorch Learn the Basics](https://docs.pytorch.org/tutorials/beginner/basics/intro.html)
- [PyTorch Recipes](https://docs.pytorch.org/tutorials/recipes/recipes_index.html) - reproducibility, checkpointing, mixed precision, and profiling.
- [Dive into Deep Learning](https://d2l.ai/) - selected chapters on MLPs, optimization, CNNs, attention, and transformers.

## Phase 4 - Transformers and LLMs

- [Hugging Face LLM Course](https://huggingface.co/learn/llm-course/en/chapter1/1) - chapters 1-3, 5-7, 10-12.
- [Hugging Face Transformers Training](https://huggingface.co/docs/transformers/training)
- [The Annotated Transformer](https://nlp.seas.harvard.edu/annotated-transformer/) - implementation-focused companion to the original architecture.
- [Attention Is All You Need](https://arxiv.org/abs/1706.03762) - primary paper.

## Phase 5 - Evaluations

- [Anthropic: Demystifying Evals for AI Agents](https://www.anthropic.com/engineering/demystifying-evals-for-ai-agents)
- [Anthropic: Define Success and Build Evaluations](https://platform.claude.com/docs/en/docs/build-with-claude/develop-tests)
- [UK AI Security Institute: Inspect](https://inspect.aisi.org.uk/)
- [OpenAI Evals](https://github.com/openai/evals)
- [EleutherAI Language Model Evaluation Harness](https://github.com/EleutherAI/lm-evaluation-harness)

## Phase 6 - RL and post-training

- [Sutton and Barto: Reinforcement Learning, 2nd ed.](http://incompleteideas.net/book/the-book-2nd.html) - selected chapters 1-3, 6, 13.
- [OpenAI Spinning Up: Key Concepts in RL](https://spinningup.openai.com/en/latest/spinningup/rl_intro.html)
- [Hugging Face Deep RL Course](https://huggingface.co/learn/deep-rl-course/en/unit0/introduction) - use for fundamentals; note that the course is in low-maintenance mode.
- [Hugging Face TRL](https://huggingface.co/docs/trl/index) - current SFT, DPO, reward-model, PPO, and GRPO tooling.
- [Learning to Summarize from Human Feedback](https://arxiv.org/abs/2009.01325)
- [Training Language Models to Follow Instructions with Human Feedback](https://arxiv.org/abs/2203.02155)
- [Direct Preference Optimization](https://arxiv.org/abs/2305.18290)
- [DeepSeekMath / GRPO](https://arxiv.org/abs/2402.03300)

## Phase 7 - ML systems

- [PyTorch Distributed Overview](https://docs.pytorch.org/tutorials/beginner/dist_overview.html)
- [PyTorch DDP Series](https://docs.pytorch.org/tutorials/beginner/ddp_series_intro.html)
- [PyTorch FSDP Tutorial](https://docs.pytorch.org/tutorials/intermediate/FSDP_tutorial.html)
- [vLLM Documentation](https://docs.vllm.ai/en/latest/)
- [Ray Train](https://docs.ray.io/en/latest/train/train.html)
- [Full Stack Deep Learning](https://fullstackdeeplearning.com/)

## Phase 8 - Interviews and communication

- [Anthropic: How We Hire](https://www.anthropic.com/careers?cid=779)
- [Google DeepMind Careers and Interview Process](https://deepmind.google/careers/)
- [Papers with Code](https://paperswithcode.com/) - locate implementations and benchmarks; verify claims in the primary paper.
