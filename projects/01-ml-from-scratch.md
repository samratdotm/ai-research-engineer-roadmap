# Project 1 - ML From Scratch

## Question

Can I explain and implement the mechanics connecting data, loss, gradients, and parameter updates?

## Build

- Linear regression with closed-form and gradient-descent solutions.
- Logistic regression with stable binary cross-entropy.
- A tiny reverse-mode automatic-differentiation engine.
- Finite-difference gradient checks.
- Synthetic data generator with controllable noise.

## Experiments

- Learning-rate sweep.
- Feature-scale sensitivity before/after normalization.
- Sample-size/noise effect on coefficient uncertainty.

## Acceptance criteria

- Tests cover shapes, gradients, convergence, and numerical stability.
- Results include plots and uncertainty where applicable.
- Report explains every equation in plain language.
- Five-minute explanation completed without notes.
