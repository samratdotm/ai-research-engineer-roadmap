# AI Research Engineer Roadmap

A 52-week, project-first curriculum for a working software engineer moving into AI evaluations, RL environments, research tools, and post-training systems.

**Start:** July 13, 2026
**Finish:** July 11, 2027
**Core commitment:** 10 hours per week
**Primary target:** Research Engineer - agent evaluations and RL environments
**Near-term roles:** AI Evals Engineer, Research Tools Engineer, RL Full-Stack Engineer, AI Reliability Engineer

## Why this roadmap exists

The goal is not to collect AI certificates or memorize vocabulary. By the end, I should be able to:

1. Explain the central ML, deep-learning, transformer, evaluation, and RL concepts without notes.
2. Implement the important mechanisms rather than only calling APIs.
3. Design controlled experiments and distinguish real model improvement from noise or infrastructure failure.
4. Build reliable training, evaluation, and serving systems.
5. publish reproducible technical work that supports a Research Engineer application.

## The 10-hour weekly cadence

- **2 hours - concepts:** lectures, reading, derivations, and concept notes.
- **4 hours - implementation:** code core mechanisms and complete exercises.
- **2 hours - experiment:** run a controlled comparison for the current phase project.
- **1 hour - communication:** write results, limitations, or a paper note.
- **1 hour - interviews:** explain concepts aloud and answer phase-specific questions.

Optional stretch work is never allowed to displace sleep, work performance, or the core ten hours.

## Definition of done

A phase is complete only when all eight conditions are met:

- [ ] I can explain the concepts without notes.
- [ ] I implemented the central mechanism.
- [ ] Automated tests pass.
- [ ] I compared against a simple baseline.
- [ ] I ran at least one controlled experiment.
- [ ] I analyzed failures rather than reporting only an average.
- [ ] I wrote what the evidence supports, including limitations.
- [ ] I passed the phase interview checkpoint at 70% or higher.
