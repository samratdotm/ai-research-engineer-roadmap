# Weekly Cadence

## Suggested calendar

| Block | Time | Work |
|---|---:|---|
| Tuesday | 90 min | Concepts and derivations |
| Wednesday | 90 min | Guided implementation |
| Thursday | 90 min | Independent implementation |
| Saturday | 3 hours | Phase project and experiment |
| Sunday | 2.5 hours | Project, writing, interview rehearsal, weekly review |

Move the blocks as needed; preserve the categories.

## Weekly outputs

- One concept note.
- One tested implementation or experiment increment.
- One committed project improvement.
- One five-minute oral explanation.
- One weekly review.

## Recovery rule

If a week falls below six hours, do not mark it complete. Carry the unfinished acceptance criteria into the next week and remove optional stretch work.

## Stretch rule

Extra hours may be used for paper reproduction, open-source contributions, or additional experiments. Do not start an unrelated application.
