# Phase 0 - Orientation

## Questions this phase must answer

- What is learned during training, and what stays fixed during inference?
- Where do data, loss, optimizer, parameters, checkpoints, and metrics fit?
- How is an agent system different from the model it calls?
- Which parts of my ADP work are ML, AI application engineering, and distributed systems?

## Study checklist

- [ ] AI, ML, deep learning, generative AI, and reinforcement learning.
- [ ] Supervised, unsupervised, self-supervised, and reinforcement learning.
- [ ] Features, labels, examples, datasets, batches, epochs, parameters, and hyperparameters.
- [ ] Training, validation, testing, inference, deployment, and monitoring.
- [ ] Model, scaffold, tool, retriever, memory, evaluator, and product workflow.

## Deliverables

1. `baseline.ipynb`: Python, NumPy, statistics, and terminology exercises.
2. `gap-assessment.md`: known, partial, unknown, and evidence for each category.
3. Ten-minute explanation: trace one ADP agent request and identify which parts are model behavior versus deterministic software.

## Exit test

Explain why an application can improve without changing model weights, and why a model can improve while the product becomes worse.
