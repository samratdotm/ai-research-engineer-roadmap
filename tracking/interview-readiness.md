# Interview Readiness

Score each category monthly from 0-4:

- **0:** cannot answer.
- **1:** recognize terminology.
- **2:** explain with help.
- **3:** explain and implement independently.
- **4:** handle tradeoffs, failures, and follow-up questions.

## Required mock loop

- [ ] 45-minute Python coding/debugging.
- [ ] 45-minute ML fundamentals.
- [ ] 45-minute experiment design and result interpretation.
- [ ] 60-minute ML/eval system design.
- [ ] 45-minute Agent Inc. or EvalLab deep dive.

## Application gate

Begin targeted applications when:

- Evaluations/experimental design reaches 3 or higher.
- Transformers/LLMs reaches 3 or higher.
- Project deep dive reaches 4.
- At least one research-style artifact is public.

Do not wait for every category to reach 4.
